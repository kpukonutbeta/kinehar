# Panduan Penggunaan

Ikuti langkah-langkah berikut untuk mulai menggunakan aplikasi KEJAR secara maksimal.

## 1. Memilih Profil Pegawai
Sebelum melakukan aktivitas apapun, Anda harus memilih profil Anda terlebih dahulu:
1. Pergi ke menu **Profil**.
2. Pilih nama Anda dari daftar pegawai yang tersedia.
3. Setelah dipilih, sistem akan menyimpan identitas Anda secara otomatis (Session).

## 2. Mengisi Kinerja Harian
Setelah profil terpilih:
1. Kembali ke halaman **Beranda**.
2. Masukkan deskripsi pekerjaan pada kolom **Uraian Tugas**.
3. Tuliskan hasil dari pekerjaan tersebut di kolom **Output/Hasil**.
4. Klik **Simpan**.

## 3. Melakukan Presensi
Jangan lupa mencatat kehadiran Anda:
1. Pilih menu **Absensi**.
2. Pastikan GPS aktif.
3. Klik **Simpan Kehadiran** pada jam yang sesuai (Pagi/Siang/Sore).

## 4. Melihat Riwayat
Anda dapat memantau data yang sudah terkirim:
- Gunakan menu **Histori** untuk melihat daftar laporan per tanggal.
- Gunakan menu **Statistik** untuk melihat grafik capaian kinerja bulanan Anda.
