# Formulir Isian Kinerja Harian

Halaman utama aplikasi digunakan untuk mengisi laporan tugas yang telah diselesaikan pada hari tersebut.

## Komponen Formulir

### 1. Header Identitas
Menampilkan nama dan NIP pegawai yang sedang aktif. Data ini diambil secara otomatis berdasarkan profil yang Anda pilih.

### 2. Pemilih Tanggal
Anda dapat memilih tanggal penugasan. Secara default, sistem akan memilih tanggal hari ini.

### 3. Uraian Tugas
Kolom teks untuk mendeskripsikan secara detail apa saja yang Anda kerjakan. 
> **Tips**: Gunakan poin-poin agar lebih mudah dibaca oleh verifikator.

### 4. Output / Hasil
Kolom ini digunakan untuk menuliskan hasil nyata dari uraian tugas di atas. Contoh: "Laporan Terkirim", "Surat Selesai Diinput", dsb.

## Cara Mengirim Laporan
1. Pastikan profil Anda sudah benar.
2. Isi **Uraian Tugas** dan **Output/Hasil**.
3. Klik tombol **"Simpan Kinerja"**.
4. Tunggu hingga muncul notifikasi keberhasilan.

::: tip Info
Data Anda akan langsung masuk ke Google Sheets verifikator secara real-time.
:::
